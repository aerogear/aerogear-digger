# AeroGear Digger Android Jenkins slave OpenShift Docker image

This directory contains the Docker source for Android Jenkins slave for AeroGear Digger.

This image is available on Docker Hub: <https://hub.docker.com/r/aerogear/jenkins-android-slave/>

## Build

To build slave execute docker build

    docker build . -t aerogear/jenkins-android-slave
